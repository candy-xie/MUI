头部

Header

Header是固定在屏幕顶部的组件。可以包含如标题和左右的功能按钮, 可以自定义颜色。

bar-light

bar-stable

bar-positive

                     <div class="bar bar-header bar-light">
                     <h1 class="title">bar-light</h1>
                     </div>
                     <div class="bar bar-header bar-stable">
                     <h1 class="title">bar-stable</h1>
                     </div>
                     <div class="bar bar-header bar-positive">
                     <h1 class="title">bar-positive</h1>
                     </div>

Sub Header

Sub Header同样是固定在顶部，只是是在Header的下面，就算没有写Header这个， Sub Header这个样式也会距离顶部有一个Header的距离。可以添加Header组件的颜色Class ( bar-stable,bar-positive 等 )，给Sub Header添加背景色。

Header

Sub Header

                     <div class="bar bar-header">
                     <h1 class="title">Header</h1>
                     </div>
                     <div class="bar bar-subheader">
                     <h1 class="title">Sub Header</h1>
                     </div>

按钮

Button

button的默认情况是按照display: inline-block来显示，宽度是随其内部文字长度增长的。 有许多不同的显示样式。

                     <button class="button button-light">button-light</button>
                     <button class="button button-stable">button-light</button>
                     <button class="button button-positive">button-light</button>

Block Button

想让button占满父元素宽度，需要加上 button-block 样式

                     <button class="button button-block button-light">button-light</button>
                     <button class="button button-block button-stable">button-light</button>
                     <button class="button button-block button-positive">button-light</button>

Different Button

控制button的大小，添加button-small使按钮变小，添加button-large使按钮变大。

                     <button class="button button-small button-light">button-small</button>
                     <button class="button button-stable">button-normal</button>
                     <button class="button button-large button-positive">button-large</button>

Outlined Button

添加button-outline使按钮只有边框。

                     <button class="button button-outline button-light">button-light</button>
                     <button class="button button-outline button-stable">button-stable</button>
                     <button class="button button-outline button-positive">button-positive</button>

Clear Button

添加button-clear使按钮只显示文字，可以当链接。

                     <button class="button button-clear button-light">button-light</button>
                     <button class="button button-clear button-stable">button-stable</button>
                     <button class="button button-clear button-positive">button-positive</button>

Icon Button

带图标的按钮，可以控制icon的位置。

                     <button class="button button-clear button-positive icon-left iconfont icon-jiantou">返回</button>
                     <button class="button button-positive icon-right iconfont icon-jiantou2">更多</button>
                     <button class="button iconfont icon-ordinaryset"></button>
                     <button class="button button-icon iconfont icon-ordinaryset"></button>

工具栏

工具栏是水平排列的button或link，用于切换视图，是一种很常用的导航方法

只有文案

tab1 tab2 tab3 
                     <div class="tabs">
                     <a href="#" class="tab-item">tab1</a>
                     <a href="#" class="tab-item">tab2</a>
                     <a href="#" class="tab-item">tab3</a>
                     </div>

只有图标

工具栏的背景色可自定义。

                     <div class="tabs tabs-positive">
                     <a href="#" class="tab-item"><i class="iconfont icon-home"></i></a>
                     <a href="#" class="tab-item"><i class="iconfont icon-my"></i></a>
                     <a href="#" class="tab-item"><i class="iconfont icon-gouwuche1"></i></a>
                     </div>

图标上，文字下

首页 我的 购物车 
                     <div class="tabs tabs-icon-top tabs-positive">
                     <a href="#" class="tab-item"><i class="iconfont icon-home"></i>首页</a>
                     <a href="#" class="tab-item"><i class="iconfont icon-my"></i>我的</a>
                     <a href="#" class="tab-item"><i class="iconfont icon-gouwuche1"></i>购物车</a>
                     </div>
